# Frontend Design Principles

- Use DaisyUI for CSS framework
- Refer to The Futur Type Manual for typography decisions
- Follow Dieter Rams / Jony Ive "less, but better" philosophy
- No Apple imitation
- Minimal, principled design
- No decorative elements that serve no function
- Typography hierarchy: one serif for headings, one sans-serif for body
- Maximum two colours plus neutrals
- Mobile-first
- Static HTML/CSS/JS only
- No frameworks unless explicitly asked
